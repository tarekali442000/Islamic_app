# Islamic App

🕌 مواقيتي – Mawaqiti:

✨ المميزات:

✅ عرض مواقيت الصلاة لأي مدينة داخل مصر 

✅ بث مباشر لإذاعات القرآن الكريم 

✅ مجموعة كاملة من الأذكار اليومية (الصباح والمساء) 

✅ أحاديث وأدعية مختارة 

✅ تصميم متجاوب يعمل على جميع الأجهزة 

✅ واجهة سهلة وبسيطة الاستخدام
